# Weather-Forecasting

# NAME: TEJASHREE H

## DESCRIPTION:- This project is a Python-based weather forecasting application that shows detailed temperature, rainfall, humidity, and seasonal trends for Mangalore. The app provides an interactive interface where users can select a date range, view historical data, analyze climate patterns, and generate weather forecasts. It also visualizes data using clean, easy-to-read graphs.

 # Key Features

 * Interactive Weather Dashboard with date selection and control buttons.

 * Temperature & Rainfall Analysis with line charts and bar graphs.

 * Season-wise Temperature Comparison (Winter, Pre-monsoon, Monsoon, Post-monsoon).

 * Monthly Rainfall Pattern Visuals to understand climate trends.

 * Humidity vs Temperature Scatter Plot for climate behavior study.

 * Shows geo-location info like latitude, longitude, and climate type.
 
 * User-friendly GUI suitable for beginners and students.
